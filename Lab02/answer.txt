Question :If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
			KEY
	Java will not allow you to do this. The reason is that both the title and the category attributes are defined as String types, and Java distinguishes overloaded constructors based on the data types, number, and order of the parameters rather than the names of the variables. Providing two constructors that both accept a single String argument would result in a duplicate method signature, leading to a compilation error.
